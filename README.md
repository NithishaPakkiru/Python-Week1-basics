# Python-Week1-basics
Beginner Python Project Using Jupyter Notebook
This project was created as part of my Week 1 Python learning journey.
I learned Python fundamentals including variables, input/output, strings, lists, and basic logic.
The project is a Personal Introduction Program that interacts with the user and displays a friendly welcome message.
The project was developed using Python and Jupyter Notebook via Anaconda.

## Setup Instructions
1. Download and install Anaconda.
2. Open Anaconda Navigator.
3. Launch Jupyter Notebook.
4. Open the `personal_introduction_program.ipynb` file.
5. Run the cells using Shift + Enter.

## Technical Details
- The program uses Python built-in functions like `input()` and `print()`.
- Variables are used to store user information.
- Strings and f-strings are used for formatted output.
- The program runs sequentially from top to bottom.
